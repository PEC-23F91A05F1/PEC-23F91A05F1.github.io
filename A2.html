HTML Hyperlinks Example

1. Basic Hyperlink

This is a basic hyperlink to an external webpage:

2. Hyperlink with Target Attribute (Open in New Tab)

Clicking this link will open the page in a new tab:

Visit Example Website (New Tab)

3. Hyperlink to an Anchor within the Same Page

This link will jump to a section further down the page:

Go to Section 2

4. Hyperlink to a Specific Section on Another Page

This link will navigate to a specific section (if the page supports it):

Visit Example Website - Section

5. Hyperlink to Email

Clicking this link will open your email client with a new email to the specified address:

Send an Email to Someone
